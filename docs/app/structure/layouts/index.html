Layouts

A XDK disponibiliza alguns layouts prontos para as aplicações. São eles:

info

Usamos a mesma tela em todos os exemplos para facilitar a identificação das diferenças entre os layouts.

DefaultLayoutComponent​

Layout padrão da XDK. Conta com uma barra superior com informações sobre o título da página, ações de login, etc. Também tem um menu lateral e pode exibir um rodapé, se configurado.

Utilização:

/src/app/app-routing.module.ts
import { DefaultLayoutComponent } from '@itix/components';export const routes: Routes = [    // ...    {        path: '',        component: DefaultLayoutComponent,        children: [            // Todos os itens aqui terão o layout aplicado.        ]    }    // ...]

SideMenuLayoutComponent​

Layout com um menu lateral que também tem informações sobre o título da página, ações de login, etc e pode exibir um rodapé, se configurado.

Utilização:

/src/app/app-routing.module.ts
import { SideMenuLayoutComponent } from '@itix/components';export const routes: Routes = [    // ...    {        path: '',        component: SideMenuLayoutComponent,        children: [            // Todos os itens aqui terão o layout aplicado.        ]    }    // ...]

BlankLayoutComponent​

Layout sem componentes adicionais.

Utilização:

/src/app/app-routing.module.ts
import { BlankLayoutComponent } from '@itix/components';export const routes: Routes = [    // ...    {        path: '',        component: BlankLayoutComponent,        children: [            // Todos os itens aqui terão o layout aplicado.        ]    }    // ...]
